# aksw-to-jekyll-rdf
## Requirements
Ruby > 2.1
Gem > 2.1
Bundler
## Installation 
```
gem install jekyll
gem install jekyll-rdf
git clone git@github.com:Simaris/aksw-to-jekyll-rdf.git
```
## Run aksw-to-jekyll-rdf
To run aksw-to jekyll-rdf change into the directory of your local repo
```
cd aksw-to-jekyll-rdf
```
and then run jekyll
```
jekyll serve
```
